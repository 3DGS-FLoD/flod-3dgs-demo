Mouse

Orbit around the focal point
RMB+Drag
Pan the camera and focal point
Scroll

Keyboard
WASD
Move camera
C
Toggle mesh cursor (intersection point)
I
Toggle info panel (FPS, camera position, etc.)
U
Toggle debug object (camera orientation)
←
Rotate camera up vector counter-clockwise
→
Rotate camera up vector clockwise
P
Toggle point-cloud mode
=
Increase splat scale
-
Decrease splat scale
O
Toggle orthographic mode
